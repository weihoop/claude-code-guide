# 合同审查要点备忘录

> 供 Claude 分析合同时参考，按合同类型列举核查要点。

---

## 通用合同审查要点（所有合同均需核查）

### 基本条款完整性
- [ ] 当事人信息是否完整（全称、统一社会信用代码/身份证号、地址）
- [ ] 标的是否明确（产品/服务的名称、规格、数量、质量标准）
- [ ] 价款或报酬是否明确（含税、不含税、支付方式、账户信息）
- [ ] 履行期限、地点、方式是否约定
- [ ] 违约责任是否对等（注意是否只约定一方违约责任）
- [ ] 争议解决条款（仲裁或诉讼、管辖法院/仲裁委）
- [ ] 合同生效条件

### 风险核查要点
- [ ] 付款条件是否有利（提前付款/货到付款的比例）
- [ ] 验收标准是否明确
- [ ] 保密条款范围是否合理
- [ ] 知识产权归属
- [ ] 不可抗力定义是否过宽
- [ ] 单方解除权条款是否有利

---

## 买卖合同

### 甲方（买方）立场
- [ ] 质量标准和验收程序是否明确
- [ ] 交货方式（FOB/CIF等）和风险转移节点
- [ ] 迟延交货的违约责任（具体赔偿比例）
- [ ] 质量保证期和保修义务
- [ ] 货物损毁灭失责任
- [ ] 退货/换货条件

### 乙方（卖方）立场
- [ ] 付款时间和条件是否明确，逾期付款违约金
- [ ] 验收期限和异议期限是否合理
- [ ] 不可预见成本上涨是否有调价机制
- [ ] 货款优先受偿（所有权保留条款）

---

## 服务合同

### 核查要点
- [ ] 服务内容和范围是否清晰（避免无限扩大）
- [ ] 服务标准（可量化的 KPI 或验收标准）
- [ ] 服务期限和续约条件
- [ ] 人员要求（是否指定特定人员）
- [ ] 甲方配合义务（否则不能免责）
- [ ] 单方提前解约的代价
- [ ] 服务成果的知识产权归属

---

## 劳动合同

### 用人单位立场
- [ ] 试用期约定（≤6个月，最多1次）
- [ ] 岗位职责描述（调岗依据）
- [ ] 保密和竞业限制条款（补偿金是否约定）
- [ ] 规章制度培训记录（作为违纪依据）
- [ ] 绩效考核与调薪条件

### 劳动者立场
- [ ] 薪资组成（基本工资/绩效/奖金比例）
- [ ] 工作时间和加班工资约定
- [ ] 社保缴纳基数
- [ ] 解除合同的补偿/赔偿条款
- [ ] 竞业限制范围是否合理，补偿金是否明确

---

## 房屋租赁合同

### 核查要点
- [ ] 出租人是否有权出租（产权人或有转租权的承租人）
- [ ] 房屋基本情况（地址、面积、用途是否一致）
- [ ] 租金及调价机制（年涨幅上限）
- [ ] 押金金额及退还条件
- [ ] 装修/改建权限及费用分担
- [ ] 租期届满后的优先续租权
- [ ] 提前解约的违约责任
- [ ] 房屋维修义务分配
- [ ] 水电煤气等费用分担

---

## 工程承包合同（建设工程）

### 发包人立场
- [ ] 工程量变更的计价机制
- [ ] 竣工验收标准和逾期违约金
- [ ] 质量保证金比例和退还条件（不超过3%，《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释（一）》法释〔2020〕25号）
- [ ] 安全责任划分
- [ ] 分包审批权

### 承包人立场
- [ ] 工程款支付节点和逾期利息
- [ ] 发包人逾期提供图纸/材料的索赔权
- [ ] 材料价格波动调整机制
- [ ] 停工损失索赔

---

## 借款合同（民间借贷）

### 核查要点
- [ ] 利率是否超过 LPR 4 倍（《民间借贷司法解释》第25条）
- [ ] 还款期限和提前还款权利
- [ ] 担保方式（保证/抵押/质押）是否完善
- [ ] 逾期利率约定（不超过合同利率的1.3倍）
- [ ] 砍头息条款（预先扣除利息无效）
- [ ] 资金用途限制

---

## 合伙/合作协议

### 核查要点
- [ ] 出资比例和出资方式（货币/实物/知识产权/劳务）
- [ ] 利润分配和亏损分担
- [ ] 决策机制（重大事项表决规则）
- [ ] 退出机制和股权回购价格计算
- [ ] 竞争禁止条款
- [ ] 关联交易的审批程序
- [ ] 合伙终止的清算程序

---

## 常见缺失条款清单

| 条款 | 重要性 | 说明 |
|------|--------|------|
| 违约责任 | 高 | 明确违约情形和赔偿标准 |
| 争议解决 | 高 | 选择仲裁或诉讼，约定管辖 |
| 合同解除 | 高 | 法定+约定解除权 |
| 通知条款 | 中 | 通知方式、送达地址、生效时间 |
| 修订程序 | 中 | 合同变更须书面同意 |
| 完整协议条款 | 中 | 本合同为最终完整协议，优先于此前沟通 |
| 不可抗力 | 中 | 定义、通知义务、影响 |
| 保密条款 | 根据需要 | 保密范围、期限、例外 |
