# 诉讼文书格式规范

> 参照《人民法院诉讼文书样式（试行）》及中国法院通用格式要求

## 通用格式规范

### 页面设置
- 纸张：A4（210mm × 297mm）
- 页边距：上下左右各 2.54cm（1 英寸）
- 行间距：建议 1.5 倍行距

### 字体规范
| 用途 | 字体 | 字号 |
|------|------|------|
| 文书标题 | 黑体 | 二号（22pt） |
| 章节标题（一、事实与理由） | 黑体 | 三号/小三（16pt） |
| 正文内容 | 仿宋_GB2312 | 小四（12pt） |
| 当事人信息 | 仿宋_GB2312 | 小四（12pt） |
| 落款 | 仿宋_GB2312 | 小四（12pt） |

### 标点与格式
- 中文文书全部使用中文标点
- 数字：涉及金额用大写汉字，括号内附阿拉伯数字（如：人民币壹拾万元整（100,000元））
- 日期格式：XXXX年XX月XX日

## 各文书类型规范

### 民事起诉状
**结构顺序：**
1. 文书标题（居中）
2. 当事人信息（原告/被告/第三人）
3. 诉讼请求（编号列举）
4. 事实与理由（分段陈述）
5. 证据清单（编号列举）
6. 此致 + 法院名称
7. 落款（具状人/代理人 + 日期）

**注意事项：**
- 原告在前，被告在后
- 有代理人的，须写明"委托代理人"及律师证号、律所
- 诉讼请求须具体、可执行（避免"适当赔偿"等模糊表述）
- 金额请求须明确数字

### 答辩状
**结构顺序：**
1. 文书标题
2. 答辩人信息（被告）、被答辩人信息（原告）
3. 答辩意见（先总后分）
4. 事实与理由
5. 此致 + 法院名称
6. 落款

**答辩期限：** 收到起诉状副本之日起 **15日** 内提交（民事），可申请延期

### 上诉状
**结构顺序：**
1. 文书标题
2. 上诉人（原审原告/被告）
3. 被上诉人（原审被告/原告）
4. 原审法院及案号
5. 上诉请求
6. 上诉理由
7. 此致 + 上级法院名称
8. 落款

**上诉期限：** 民事案件判决书送达之日起 **15日**，裁定书 **10日**

### 劳动仲裁申请书
**结构顺序：**
1. 文书标题
2. 申请人（劳动者）
3. 被申请人（用人单位，须写统一社会信用代码）
4. 仲裁请求（编号列举，含金额）
5. 事实与理由
6. 证据清单
7. 此致 + 劳动争议仲裁委员会名称
8. 落款

**仲裁时效：** 劳动争议发生之日起 **1年**（特殊情形：劳动关系存续期间拖欠工资不受时效限制）

**管辖：** 劳动合同履行地或用人单位所在地劳动争议仲裁委员会

### 强制执行申请书
**结构顺序：**
1. 文书标题
2. 申请执行人（胜诉方）
3. 被执行人（败诉方）
4. 申请执行的依据（生效判决书/仲裁裁决书编号、生效日期、主要内容）
5. 申请执行的事项（须与生效文书一致）
6. 被执行人财产线索（银行账户、房产、车辆等）
7. 此致 + 法院执行局
8. 落款

**申请期限：** 生效文书确定的履行期间届满后 **2年** 内申请

## 当事人信息填写规范

### 自然人
| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 姓名 | 必填 | 与身份证一致 |
| 性别 | 必填 | |
| 出生日期 | 建议填写 | 格式：XXXX年XX月XX日 |
| 民族 | 建议填写 | |
| 住所地 | 必填 | 与身份证地址或实际居住地 |
| 联系电话 | 必填 | |
| 身份证号 | 部分法院要求 | 建议填写 |

### 法人（公司）
| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 名称 | 必填 | 工商登记全称 |
| 统一社会信用代码 | 必填 | 18位 |
| 住所地 | 必填 | 注册地址 |
| 法定代表人 | 必填 | 姓名 + 职务 |
| 联系电话 | 建议填写 | |

## 常见错误清单

| 错误类型 | 常见表现 | 正确做法 |
|---------|---------|---------|
| 当事人信息不完整 | 缺少身份证号/统一社会信用代码 | 按规范完整填写 |
| 诉请不具体 | "赔偿损失若干" | 明确金额或计算方式 |
| 法院名称有误 | 级别/管辖错误 | 核实管辖法院 |
| 证据目录与正文不符 | 正文提及的证据未列入目录 | 保持一致 |
| 落款信息缺失 | 无具状人签名 | 自然人亲笔签名，公司加盖公章 |
