# compliance-check — 企业合规审查

## 触发词
合规审查、合规风险、企业合规、劳动合规、用工合规、数据合规、合规报告

## 功能说明
针对企业不同合规领域进行系统性风险排查，生成含高/中/低风险分级的合规审查报告 Word 文档。

## 审查领域

| 参数 | 领域 | 主要内容 |
|------|------|---------|
| `labor` | 劳动用工合规 | 劳动合同、规章制度、工时工资、社保公积金 |
| `data` | 数据安全合规 | 个人信息保护、数据分���分级、隐私政策 |
| `contract` | 合同管理合规 | 合同签订审批、履行管控、归档管理 |
| `general` | 综合合规 | 以上所有领域综合审查 |

## 使用方式

### 步骤一：收集企业基本信息
向 Claude 提供：
1. 企业名称、规模、行业
2. 当前制度文件清单（如有）
3. 具体合规关注点

### 步骤二：Claude 生成审查 JSON
Claude 根据核查清单逐项分析，输出结构化 JSON，保存为 `compliance_result.json`。

### 步骤三：生成 Word 报告
```bash
python3 ~/.claude/skills/compliance-check/scripts/generate_compliance.py \
  --area labor \
  --input compliance_result.json \
  --output 劳动用工合规审查报告.docx
```

## Word 输出格式
1. 审查概况（企业信息、审查范围、审查日期）
2. 合规问题清单（按高/中/低风险分级，表格形式）
3. 整改建议（按优先级排序）
4. 法律依据汇总
5. 律师声明 + 落款

## 风险分级标准
- **高风险**：存在违法违规行为，可能受到行政处罚或引发诉讼
- **中风险**：制度存在漏洞，建议 3 个月内完善
- **低风险**：流程不规范，建议逐步优化

## 法条引用规则（必须遵守）

### 法条库优先
引用法条时，**优先**从 `~/.claude/skills/shared/references/法条库.json` 精确匹配，原文复制 `全文` 字段，并在条文末尾标注 `[库中已核实]`。

库中没有的条文，按以下三级确定性协议处理：

### 三级确定性标注协议

| 级别 | 含义 | 处理方式 |
|------|------|---------|
| `VERIFIED` | 可信度高、常用条文 | 直接引用，不加特殊标注 |
| `UNCERTAIN` | 不确定条文号或内容 | 在条文末尾加 `[待核实]`，禁止编造全文 |
| `OBSOLETE` | 已废止法律 | 禁止引用废止法律名称，必须替换为现行法 |

### 禁止行为
- **禁止**引用以下已废止法律名称（必须替换为括号内的现行法）：
  - ~~婚姻法~~ → 《民法典》第五编婚姻家庭
  - ~~合同法~~ → 《民法典》第三编合同
  - ~~侵权责任法~~ → 《民法典》第七编侵权责任
  - ~~物权法~~ → 《民法典》第二编物权
  - ~~继承法~~ → 《民法典》第六编继承
  - ~~担保法~~ → 《民法典》相关编章
- **禁止**生成超出法律实际条数的条文号（如《劳动合同法》正文止于第98条，《民法典》止于第1260条）
- **禁止**在不确定时编造条文全文

### JSON 输出格式要求
在输出 JSON 时，每条法条必须携带来源字段：
```json
{
  "法条": "《劳动合同法》第47条：经济补偿按劳动者...",
  "_来源": "LIBRARY",
  "_确定性": "VERIFIED"
}
```
或：
```json
{
  "法条": "《劳动合同法》第XX条（条文号待核实）",
  "_来源": "AI_GENERATED",
  "_确定性": "UNCERTAIN"
}
```

其中 `_来源` 取值：`LIBRARY`（来自法条库）或 `AI_GENERATED`（AI 生成，需核实）。

## 案例参考（可选）

分析案件时，可参考中国法院年度案例库中的真实判例：

1. 读取 `~/.claude/skills/shared/references/案例索引.json`
2. 按当前案件的领域/案由/关键词，检索相似案例
3. 如需案例详情，读取 `~/.claude/skills/shared/references/案例全文/{年份}/{册号}-{领域}/{序号}.txt`
4. 引用时标注：`[参考案例：{案例标题}，{裁判书字号}]`

注意：案例仅供参考，不构成判决依据。各地法院裁判尺度可能不同。
