# evidence-organize — 证据整理

## 触发词
证据目录、质证意见、整理证据、证据清单、出具质证、证据整理、质证

## 功能说明
生成标准格式的证据整理文书，包含：
- 己方证据目录（序号、证据名称、证明目的、来源、页码）
- 对方证据质证意见（真实性、合法性、关联性三性意见）

## 使用方式

### 步骤一：填写证据信息 JSON
```bash
cp ~/.claude/skills/evidence-organize/references/evidence_template.json ./证据信息.json
# 编辑证据信息.json，填入实际证据
```

### 步骤二：生成证据文书
```bash
python3 ~/.claude/skills/evidence-organize/scripts/generate_evidence.py \
  --input 证据信息.json \
  --output 证据目录及质证意见.docx
```

### Claude 辅助模式
将证据清单描述告知 Claude，Claude 自动整理为 JSON 并生成文书。

## Word 输出格式

### 表1：己方证据目录
| 序号 | 证据名称 | 证明目的 | 证据来源 | 页码 |
|------|---------|---------|---------|------|

### 表2：对方证据质证意见
| 序号 | 证据名称 | 真实性 | 合法性 | 关联性 | 质证意见 |
|------|---------|-------|-------|-------|---------|

## 三性说明
- **真实性**：证据是否真实存在，有无伪造、变造
- **合法性**：取证方式是否合法，形式是否符合要求
- **关联性**：与本案待证事实是否有关联

## JSON 输入格式
```json
{
  "案件名称": "原告张三诉被告李四合同纠纷案",
  "案号": "（2025）粤01民初XXX号",
  "出具方": "原告代理律师",
  "己方证据": [
    {
      "序号": 1,
      "证据名称": "购销合同",
      "证明目的": "证明双方存在合同关系及货款金额",
      "来源": "当事人提供",
      "页码": "1-5"
    }
  ],
  "对方证据": [
    {
      "序号": 1,
      "证据名称": "收货单",
      "真实性": "有异议",
      "合法性": "无异议",
      "关联性": "有异议",
      "质证意见": "该收货单无当事人签字，真实性存疑；即便真实，亦无法证明货物质量合格"
    }
  ]
}
```
