# 自媒体 Skills

覆盖日常选题、写脚本、写文案、数据分析等自媒体工作的完整 Skill 系统。

## 功能简介

自媒体 Skills 是专门为自媒体工作者设计的工具集，覆盖从选题到数据分析的完整工作流。

### 核心特点

- ✅ **选题助手**: AI 辅助选题、热点追踪
- ✅ **脚本创作**: 视频脚本、文案生成
- ✅ **数据分析**: 内容表现、用户画像
- ✅ **工作流自动化**: 减少重复性工作

### 适用场景

| 场景 | 说明 |
|------|------|
| **视频创作** | 短视频、Vlog、教程脚本 |
| **图文创作** | 公众号、小红书、知乎 |
| **数据运营** | 内容分析、优化策略 |
| **团队协作** | 内容规划、任务分工 |

## 自媒体工作流

### 1. 选题阶段

#### 热点追踪

```
用户: "分析今天的热搜话题，找出适合我的内容方向"
[我的领域：科技、互联网]

Claude:
今日热点分析：
1. AI 大模型发布会（相关度：高）
   - 角度：技术解读
   - 受众：科技爱好者
   - 预期热度：⭐⭐⭐⭐

2. 新款手机发布（相关度：中）
   - 角度：评测对比
   - 受众：数码用户
   - 预期热度：⭐⭐⭐

推荐选题：AI 大模型的实际应用场景分析
```

#### 选题脑暴

```
用户: "围绕'AI 工具'，给我10个选题方向"

Claude:
1. 《10个提升效率的 AI 工具推荐》
2. 《AI 工具如何改变内容创作？》
3. 《程序员必备的 5 个 AI 辅助工具》
4. 《AI 工具的隐私风险，你注意了吗？》
5. 《从入门到精通：AI 工具学习路线》
...
```

### 2. 内容创作阶段

#### 视频脚本创作

```
用户: "写一个 3 分钟的短视频脚本，主题：10个 AI 工具"

Claude:
【开场】(0-15秒)
"今天给大家分享 10 个宝藏 AI 工具..."

【主体】(15秒-2分30秒)
工具1：ChatGPT
- 功能：对话、写作
- 演示：生成一篇小红书文案
- 点评：最常用的 AI 助手

工具2：Midjourney
- 功能：AI 绘画
...

【结尾】(2分30秒-3分)
"关注我，每天分享 AI 干货..."
```

#### 图文文案创作

```
用户: "写一篇小红书笔记：AI 工具提升效率"

Claude:
标题：💡 这 5 个 AI 工具，让我效率翻倍！

✨ 写作不再头疼
ChatGPT 帮我：
• 起草大纲
• 优化文案
• 翻译润色

🎨 设计小白也能出图
Midjourney 轻松搞定...

[配 5 张信息图]
```

### 3. 数据分析阶段

#### 内容表现分析

```
用户: "分析我最近 10 篇文章的数据"
[提供数据：阅读量、点赞、评论、转发]

Claude:
数据洞察：

📊 整体表现
- 平均阅读：1.2万
- 互动率：3.5%
- 最佳发布时间：晚上 8-10 点

🔥 爆款特征
- 标题带数字：阅读量 +40%
- 图文混排：完读率 +25%
- 实用教程类：转发率最高

💡 优化建议
1. 增加教程类内容占比
2. 优化标题，使用数字
3. 保持晚间发布时间
```

#### 用户画像分析

```
用户: "分析我的粉丝画像"
[提供评论数据、互动记录]

Claude:
粉丝画像：

👥 基础属性
- 年龄：25-35 岁为主（60%）
- 性别：男性占 65%
- 地域：一线城市（70%）

💼 职业分布
- 互联网从业者：40%
- 学生：25%
- 其他：35%

🎯 兴趣偏好
- AI 工具：最感兴趣
- 编程技术：次之
- 职业发展：关注度高

内容建议：
聚焦 AI 工具 + 编程技术的交叉内容
```

## 工作流自动化

### 批量生成

```
用户: "生成 5 个视频脚本大纲，主题：AI 工具"

Claude: [批量生成]
1. 《AI 写作工具对比》
2. 《AI 绘画入门教程》
3. 《AI 辅助编程实战》
4. 《AI 工具的隐私问题》
5. 《免费 AI 工具推荐》

[每个大纲包含：开场、主体、结尾]
```

### 跨平台改编

```
用户: "将这篇文章改编为：小红书笔记、Twitter 推文、视频脚本"

Claude:
✅ 小红书版本
- 标题：emoji + 数字
- 正文：短句 + 图片
- 长度：500 字

✅ Twitter 版本
- 拆分为推文串（5 条）
- 每条 <280 字符
- 添加话题标签

✅ 视频脚本版本
- 时长：3 分钟
- 分镜：10 个场景
- 字幕：逐句标注
```

## 最佳实践

### 1. 建立素材库

```
内容素材分类：
- 选题库：100+ 选题方向
- 案例库：成功案例、失败教训
- 数据库：行业数据、趋势报告
- 模板库：标题模板、文案模板
```

### 2. 数据驱动创作

```
创作前：
- 分析历史数据
- 确定受众偏好
- 选择最佳话题

创作后：
- 追踪内容表现
- 分析用户反馈
- 迭代优化策略
```

### 3. 保持原创性

```
AI 辅助，而非替代：
✅ 用 AI 生成大纲和灵感
✅ 用 AI 优化文案和结构
✅ 自己添加观点和案例
✅ 保持个人风格和温度
```

## 常见问题

### AI 生成的内容太通用

**解决方法**:
1. 提供详细的背景信息
2. 加入个人经验和案例
3. 多次迭代优化
4. 添加独特的视角

### 如何避免同质化

**建议**:
- 建立个人风格和标签
- 深度而非宽度（聚焦细分领域）
- 结合个人经历
- 定期创新内容形式
